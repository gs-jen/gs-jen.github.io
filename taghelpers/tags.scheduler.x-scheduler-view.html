tag x-scheduler-view in Scheduler

Brief

The definition of one view inside of the scheduler tag.

Declaration
<x-scheduler-view name="text" title="text" view-type="day|week|workWeek|month|agenda|timeline|timelineWeek|timelineWorkWeek|timelineMonth" all-day-slot="true|false" column-width="number" editable="true|false" no-create="true|false" no-change="true|false" no-delete="true|false" start-time="" end-time="" event-height="number" major-tick="none|day|week" minor-tick-count="number" selected="true|false" selected-date-format="text" selected-short-date-format="text" show-work-hours="true|false" work-week-start="number" work-week-end="number" />

Parameters
name

The (internal) name of the view.
title

The user-friendly title of the view displayed by the scheduler.
view-type

The type of the view. The built-in views are: "day", "week", "workWeek", "month", "agenda", "timeline", "timelineWeek", "timelineWorkWeek" and "timelineMonth".
all-day-slot

If set to true the scheduler will display a slot for "all day" events.
column-width

The width of the table columns in timeline views. Value is treated as pixels.
editable

Either false (default value) to make the view not editable or nothing.

To make the view editable use x-scheduler-editable tag.
start-time

The start time of the view. The scheduler will display events starting after the startTime.
end-time

The end time of the view. The scheduler will display events ending before the endTime.
event-height

The height of the scheduler event rendered in month and timeline views.
major-tick

The number of minutes represented by a major tick for day and week view.
minor-tick-count

The number of time slots to display per major tick.
selected

If set to true the view will be initially selected by the scheduler widget.

If more than one view is selected then last of them will prevail.
selected-date-format

The format used to display the selected date.

For Kendo date formats see Date formatting in Kendo UI documentation

Contains two placeholders - "{0}" and "{1}" which represent the start and end date displayed by the view e.g. {0:dd-MM-yyyy}.
selected-short-date-format

The format used to display the selected date when viewport is narrow, and the date is displayed in short ("d") format by default.
show-work-hours

If set to true the view will be initially shown in business hours mode.

The showWorkHours option is supported when views.type is set to "day" or "week".
work-week-start

The start of working week (index based).
work-week-end

The end of working week (index based).

Details

Also you can apply the following templates to a view (and to a whole x-scheduler as well). If there is no special note about template parameter, the template parameter is an event object (for the details on the event object see SchedulerTask documentation).

Template name

Description

allDayEventTemplate

The template used to render the "all day" scheduler event.

allDaySlotTemplate

The template used to render the all day slot cell.

The template argument has the following parameters:
date - represents the slot date.
resources() - returns the relevant resources for the current slot.

dateHeaderTemplate

The template used to render the date header cells.

The template argument has the following parameters:
date - represents the major tick date.

dayTemplate

The template used to render the day slots in month view.

The template argument has the following parameters:
date - represents the major tick date.

eventDateTemplate

The template used by the agenda view to render the date of the scheduler events. The template used to render the date header cells.

The template argument has the following parameters:
date - represents the event date.

eventTemplate

The template used by the view to render the scheduler events.

eventTimeTemplate

The template used by the agenda view to render the time of the scheduler events.

majorTimeHeaderTemplate

The template used to render the major ticks.

The template argument has the following parameters:
date - represents the major tick date.

minorTimeHeaderTemplate

The template used to render the minor ticks.

The template argument has the following parameters:
date - represents the minor tick date/time.

slotTemplate

The template used to render the time slot cells.

The template argument has the following parameters:
date - represents the slot date/time. resources() - associated resources.
