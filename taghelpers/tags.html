Tag Extension Library

What are Razor-tags

The razor tags are custom tags designed to simplify creating of the pages based on Kendo UI and to separate business logic and presentation better.

How to Start Using Tags?

1. Add Kendo UI Libraries

Add .bowerrc file to the root of your ASP.NET MVC Core project.

Put the following content into .bowerrc file

.bowerrc content [show]

.bowerrc content [hide]

{
    "directory": "wwwroot/lib"
}

Add bower.json file to the root of your ASP.NET MVC Core project.

Put the following code into bower.json

bower.json content [show]

bower.json content [hide]

{
    "name": "asp.net",
    "private": true,
    "dependencies": {
        "Kendo.UI": "http://asp.gehtsoftusa.com/nuget/bower/kendo.ui.2018.1.221.zip"
    }
}

In the Visual Studio, right-click on bower.json file and choose "Restore Packages".

2. Add Kendo UI to Layout page

In the _Layout.cshtml page of your project add the following code into the header section of the page:

Code Example [show]

Code Example [hide]

<link href="~/lib/Kendo.UI/css/kendo/kendo.common.min.css" rel="stylesheet">
<link href="~/lib/Kendo.UI/css/kendo/kendo.default.min.css" rel="stylesheet">
<link href="~/lib/Kendo.UI/css/kendo/kendo.default.mobile.min.css" rel="stylesheet">
<script src="~/lib/Kendo.UI/script/jquery.min.js"></script>
<script src="~/lib/Kendo.UI/script/jquery.cookie.min.js"></script>
<script src="~/lib/Kendo.UI/script/jszip.min.js"></script>
<script src="~/lib/Kendo.UI/script/kendo.all.min.js"></script>

If you want to make theme configurable, use the following code:

Code Example [show]

Code Example [hide]

//the string variable themeName keeps the name of the chosen Kendo Theme
@if ((themeName) == "bootstrap")
{
    <link href="~/lib/Kendo.UI/css/bootstrap.min.css" rel="stylesheet">
    <link href="~/lib/Kendo.UI/css/kendo/kendo.common-bootstrap.min.css" rel="stylesheet">
}
else
{
    <link href="~/lib/Kendo.UI/css/kendo/kendo.common.min.css" rel="stylesheet">
}
<link href="~/lib/Kendo.UI/css/kendo/kendo.@(themeName).min.css" rel="stylesheet">
<link href="~/lib/Kendo.UI/css/kendo/kendo.@(themeName).mobile.min.css" rel="stylesheet">
<script src="~/lib/Kendo.UI/script/jquery.min.js"></script>
<script src="~/lib/Kendo.UI/script/jquery.cookie.min.js"></script>
<script src="~/lib/Kendo.UI/script/jszip.min.js"></script>
<script src="~/lib/Kendo.UI/script/kendo.all.min.js"></script>

Alternatively, you can use x-lib-includes tag.

3. Add Tags Support

Add Gehtsoft.TagHelpers package for the Gehtsoft private repository.

In the _ViewImports.cshtml page of your project add the following code:

Code Example [show]

Code Example [hide]

#@addTagHelper *, Microsoft.AspNetCore.Mvc.TagHelpers
#@addTagHelper *, Gehtsoft.TagHelpers
