tag x-grid in Grid

Brief

The grid.

Declaration
<x-grid id="text" var="text" editable="none|incell|inline" selectable="true|false" sortable="true|false" filterable="true|false" pageable="true|false" height="number" show-header="true|false" />

Parameters
id

The identifier of the grid.
var

The variable to place kendoGrid object during the initialization.
editable

The value indicating whether the grid data can be edited. The attribute can take one of the following values: none, incell (edit each value individually), inline (edit whole row via command).
selectable

The flag indicating whether a row of the grid may be selected.
sortable

The flag indicating whether the grid can be sorted.
filterable

The flag indicating whether the grid can be filtered.
pageable

The flag indicating whether the grid data are paginated.
height

The height of the grid in pixels or percents.
show-header

The flag indicating whether the header needs to be displayed.

Details

Use x-grid-column tags inside this tag to define grid columns.

Use x-server-datasource tags inside this tag to define grid data source.

All attributes described in x-base base tag can also be applied to this tag.

The grid supports following events:

change, dataBound, byname beforeEdit, byname edit, byname cellClose, byname cancel, byname save, byname remove, byname navigate, byname page, byname filter,

To define a toolbar, use custom script byname toolbar. The content of the script is not expected to be a function body, it will be put into toolbar property initialization "as is".

See also about toolbar.

See also Grid Overview.
