tag x-color-picker in Form

Brief

The color picker form control.

Declaration
<x-color-picker palette="websafe|basic|custom" icon="text" hue-buttons="true|false" clear-button="true|false" preview="true|false" palette-columns="number" tile-size="number" tile-width="number" tile-height="number" var="text" />

Parameters
value

A bound value of the control may be an integer, a string in HTML color format (e.g. #rrggbb where rr, gg and bb are red, green and blue components in a hexadecimal 00-ff format, or a System.Drawing.Color object.
palette

The palette.

If no palette set, a hue picker is used.

Predefined websafe (232 colors) or basic (16 colors) palettes may be used.

Also, a custom palette as a comma-separated list of colors may be set, e.g. #ff0000,#00ff00,#0000ff.
icon

The icon to be displayed on the control. See also Kendo UI icons and
hue-buttons

The flag which turns apply and cancel buttons at hue selector on or off.
clear-button

The flag sets whether the clear button will exist.
preview

The flag sets whether the preview will appear on hue selector.
palette-columns

The number of columns for palettes except hue.
tile-size

A size of a square tile for all palettes except hue. The default size is 14.
tile-width

The width of a rectangular tile for all palettes except hue. The default size is 14.
tile-height

The height of a rectangular tile for all palettes except hue. The default size is 14.
var

The name of the jscript variable to put an associated kendo object to.

Details

All attributes described in x-form-control base tag can also be applied to this tag.

See also Color Picker Overview.

Use x-message tag to set custom text for the messages.
