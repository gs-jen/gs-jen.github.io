tag x-button-group in Common

Brief

Declaration
<x-button-group id="text" selected-button="number" multi-select="true|false" disabled="true|false" var="text" items="IEnumerable<IButtonGroupItem>" />

Parameters
id

The unique identifier of the tag.
selected-button

The index of the initially selected button.
multi-select

The flag indicating whether the button group is a multi-select.
disabled

The flag indicating whether the whole control is disabled.
var

The name of the variable to store kendoButtonGroup object.
items

A collection of the button group items. Alternatively, you can use x-button-group-item tags to specify the button group content.

Details

See also ButtonGroup Overview.

All attributes described in x-base base tag can also be applied to this tag.
