class SchedulerTask

Brief

The base class for scheduler tag data

Details

The scheduler task is a C# object used to handle, serialize to Json and deserealize from Json the event (aka tasks) objects used by x-scheduler.

Use Newtonsoft.Json to serialize the object into Json and back. Please note that such conversion requires specifying two type converters RecurrenceRuleJsonConverter and RecurrenceExceptionJsonConverter, e.g.

Serializing into Json [show]

Serializing into Json [hide]

IEnumerable<SchedulerTask> Tasks;
...
JsonConvert.SerializeObject(Tasks, Formatting.Indented, new RecurrenceRuleJsonConverter(), new RecurrenceExceptionJsonConverter()).

The json object required by the scheduler control has at least the following fields:

Name

Type

Description

id

any

The unique event identifier.

title

string

The name of the event.

description

string

The detailed description of the event.

start

date

The date and time when the event starts.

For recurrent event the date is the date of the first occurrence and the time is the time when each occurrence starts.

end

date

The date and time when the event ends.

Note: This value must be set even if the event is marked as all day event.

recurrenceRule

string

In case the event is recurrent, the field contains a recurrence rule in iCal format.

See section 3.3.10 of RFC 5545 for details.

recurrenceException

string

In case there are some dates are excepted from recurrence, the field contains a comma-separated list of these dates in ISO format

isAllDay

boolean

The flag indicating that the event is an all day long event.
