tag x-script in Common

Brief

A script.

Declaration
<x-script role="script|global|click|error|change|select|databound|template|sync|byname" script-role-name="text" parameter-name="text" />

Parameters
role

The script role.

script

This is the default value. The script text inside will be added to the document onload event.

global

The script text inside will be added to global script definition on the page.

template

The script text inside is a custom template for the specific control element.

...

The script is attached to corresponding event of the tag into which the script is enclosed. If the tag does not support scripts with the role specified, a error will be raised.

The code inside of the tag will be enclosed into function (parameter) { tag content }.
role-name

The name for byname and template roles.
parameter-name

The name of the function parameter(s). The default value is e.

Details

For better IDE compatibility the whole content of the tag may be enclosed into <script> tag.

For templates, you can also use x-template for automatic creation of the template function on the base of HTML code.
