class InnerFormModel&lt;TM,TR&gt;

Brief

View model for inner (embedded) forms

Parameters
TM

The class of the inner form model.
TR

The class of the inner form response.

Details

All the pages that consists of an inner (embedded) forms must have the model derived from this class.

This class is used to:

1) Pass all information required to proper work of the tags in the embedded form.

2) Allows creation of the form where the model and the response object are separated.

If you prefer MVC-style (when model and response object are mixed-up), use Response part of the only.

The type defined in the Gehtsoft.HtmlHelpers.dll assembly. The namespace is Gehtsoft.TagHelpers.
