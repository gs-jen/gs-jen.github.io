class control

Brief

Common properties for all control definitions

Details

See also button, checkbox, dateinput, editable-select, label, numericinput, password, select, static, textarea and textinput for details about properties associated with specific type of the control.

Public Properties

autofocus

The flag indicating whether the control must be focused when the form is displayed.

buttons

An array of buttons to be added after the control into the same cell of the layout

change

The function to be called when the value is changed.

class

The space separated list of classes to apply

disabled

The flag indicating whether the control is disabled.

id

The control identifier.

keydown

The function to be called when key is pressed when the control is in focus

keyup

The function to be called when key is released when the control is in focus

label

The control label to be displayed.

name

The name of the control

placeholder

The text to be displayed in place of an empty control value.

readonly

The flag indicating that the control is read only.

style

The semicolon separated list of styles to apply.

suppresslabel

The flag indicating whether the label must not be displayed and the layout space for the label must not be reserved.

tooltip

The tooltip to display

type

The type of the control

validators

The array of the control validators

visible

The flag indicating whether control is visible

width

The width of the control.
