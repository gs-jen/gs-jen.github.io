tag x-server-transport in Transport

Brief

The definition of the transport.

Declaration
<x-server-transport read-url="url" create-url="url" update-url="url" destroy-url="url" result-type="text" read-method="text" update-method="text" />

Parameters
read-url

The URL of the action to read one or more objects.
create-url

The URL of the action to create an object.
update-url

The URL of the action to update an object.
destroy-url

The URL of the action to destroy an object.
result-type

The type of the result, e.g. xml, json or jsonp.
read-method

The method for read requests e.g. GET, POST
update-method

The method for all data modification requests e.g. GET, POST

Details

A parameterMap function may be attached to the transport to add additional parameter to the requests using x-script tag with role equal to byname and the role name parameterMap. Alternatively, use x-server-parameter to define individual additional parameters.

The tag may be used inside of x-server-datasource tag only.
