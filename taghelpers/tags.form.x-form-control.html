tag x-form-control in Form

Brief

The common attributes for all form controls.

Declaration
<x-form-control id="text" label="text" tooltip="text" placeholder="text" suppress-label="true|false" auto-focus="true|false" disabled="true|false" hidden="true|false" in-validation-name="text" whole-row="true|false" for="ModelExpression" name="text" value="text" read-only="true|false" width="text" />

Parameters
id

The control identifier.
label

The control label.
tooltip

The control tooltip.
placeholder

The control placeholder (the text to be shown when no data is selected);
suppress-label

The flag indicating whether the label must not be displayed.
auto-focus

The flag indicating whether the control must get the focus when the page is opened.
disabled

The flag indicating whether the control is disabled.
hidden

The flag indicating whether the control is not displayed.
in-validation-name

The name of the control for reference function in the validation (see x-validate). If the value is not set, the value of the name attribute is used.
whole-row

The flag indicating whether the control takes both columns in two-column form layout.
for

The model expression that is bound to the control. If this attribute is set, name and value attributes must not be used.
name

The control name.
value

The control value.
read-only

The flag indicating whether the control is read-only.
width

Width of the control in pixels or in percents.

Details

This is a pseudo-tag and cannot be used in actual templates. Instead, all attributes described in this tag may be used in the tags derived from this tag.

All attributes described in x-base base tag can also be applied to this tag.
