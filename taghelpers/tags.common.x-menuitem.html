tag x-menuitem in Common

Brief

The menu item without the target defined.

Declaration
<x-menuitem when-logged-in="true|false" permissions="list" id="text" name="text" sprite="text" image-url="url" image-width="number" image-height="number" image-alt="text" />

Parameters
when-logged-in

The flag indicating whether the command is available depending on the logged in status.

If the value is true the command is available only when a user is logged in. If the value is false the command is available only when a user is not logged in. If the attribute is not set, the command is available always.
permissions

The list of comma-separated permissions.

If the list of permissions is set, the command will be displayed only if the any of permissions are granted to the current user.
id

The menu item identifier.
name

The menu item name.
sprite

The menu sprite name associated to be used as a menu item icon.

See also Kendo UI icons and Bootstrap glyphicons for available icons.
image-url

The URL of the image to be used as a menu item icon.
image-width

The width of the image in pixels.
image-height

The height of the image in pixels.
image-alt

The alt text for the image.

Details

Note that, all attributes described in this tag may also be used in the following tags: x-menuitem-url, x-menuitem-action, x-menuitem-submenu

The item has a select event which is activated when the user clicks on the event. Use x-script to provide a function handler for the event.
