tag x-link-guard in Form

Brief

The tag that protected leaving the form via link if data is changed.

Declaration
<x-link-guard title="text" message="text" go-button="text" no-go-button="text" />

Parameters
title

The title for popup.
message

The message to be displayed.
go-button

The text on the button that allows leaving of the form.
no-go-button

The text on the button that forbids leaving of the form.

Details

If this tag is used inside a form, all links created using x-button or x-link tags will first check whether the form data are changed and will show a warning popup before leaving the page.

If you want you custom navigating control to be also covered by the link guard protection, navigate via calling a script tagHelpers_formNavigateHelper.
