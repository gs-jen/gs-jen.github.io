tag x-numeric-edit in Form

Brief

The numeric editor

Declaration
<x-numeric-edit size="number" var="text" format="text" spinners="true|false" spin-min="number" spin-max="number" spin-step="number" />

Parameters
size

This size of the control in characters.
var

The name of the variable to put kendoNumericTextBox
format

For Kendo date formats see number formatting in Kendo UI documentation
spinners

The flag indicating whether spinners (up/down arrows) needs to be displayed.
spin-min

The minimum spinner value.
spin-max

The maximum spinner value.
spin-step

The spinner step.

Details

All attributes described in x-form-control base tag can also be applied to this tag.

See also NumericTextBox Overview.
