tag x-notification in Common

Brief

The notification.

Declaration
<x-notification id="text" var="text" target="text" allow-hide-after="number" auto-hide-after="number" height="number" width="number" hide-on-click="true|false" disable-animation="true|false" stacking="up|right|left|down" pinned="true|false" position-left="number" position-top="number" position-right="number" position-bottom="number" />

Parameters
id

The tag identifier.
var

The name of the variable to which the associated kendoNotification object is stored.
target

If notifications need to be pushed into a specific tag, the identifier of such tag must be set here with '#' prefix.
allow-hide-after

Time in milliseconds after which the user can hide the notification.
auto-hide-after

Time in milliseconds after which the notification is automatically hidden.
height

The height of a notification.
width

The width of a notification.
hide-on-click

The flag indicating whether notification will be hidden on click.
disable-animation

The flag indicating whether the animation needs to be disabled.
stacking

The direction in which notification will be stacked.

The attribute can take one of the following values: up, right, left, down
pinned

Determines whether the popup notifications will move together with the other page content during scrolling.
position-left

Determines the pixel position of the first popup notification with regard to the viewport's left edge.
position-top

Determines the pixel position of the first popup notification with regard to the viewport's top edge.
position-right

Determines the pixel position of the first popup notification with regard to the viewport's right edge.
position-bottom

Determines the pixel position of the first popup notification with regard to the viewport's bottom edge.
