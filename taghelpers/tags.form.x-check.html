tag x-check in Form

Brief

A checkbox.

Declaration
<x-check/>

Details

The control supports following event scripts: click.

Please note that the checkbox label behavior is different from the labels for other controls. For other controls the label is displayed at left or above (depending of the form mode) of the control but for the checkbox, the label is a part of the control.

Therefore, to make a control properly aligned with other controls, it generates an additional empty label. However, in some situations this label is not desirable (e.g. if checkbox is placed under another control inside of a form group). In this case use the suppress-label attribute to suppress generation of this label.

All attributes described in x-form-control base tag can also be applied to this tag.
