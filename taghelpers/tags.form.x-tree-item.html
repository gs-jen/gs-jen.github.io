tag x-tree-item in Form

Brief

An item inside of tree view.

Declaration
<x-tree-item id="text" value="text" css-sprite="sprite" image-url="url" item-link="url" />

Parameters
id

Item identifier.
value

Item value (text) to be displayed.
css-sprite

A sprite id to be used as the item icon.

See also Kendo UI icons and Bootstrap glyphicons for available icons.
image-url

Url of image to be used as the item icon.
item-link

The Url to navigate if the item is clicked.

Details

The tag may be used inside of x-tree-control and x-tree-select items to set the data locally. The tag may contains other x-tree-item tags.
