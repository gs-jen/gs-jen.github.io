tag x-base in Common

Brief

The base class for all visual tags in the library.

Declaration
<x-base class="classes" style="css styles" x-class="x-classes" />

Parameters
class

The list of space separated CSS classes.
style

The list of semicolon separated CSS styles.
x-class

The list of space separated x-classes classes.

Details

This is a pseudo-tag and cannot be used in actual templates. Instead, all attributes described in this tag may be used in the following tags derived from this tag.

What is x-class?

The x-class is a simulation of multiple class inheritance. The list of x-classes are stored in the ./Content/Styles/x-styles.xml file. The dictionary is a plain xml file with styles root tag with one or more style tag inside.

Each style tags has three attributes:

tag the x-tag to which the x-style is applied to

name the name of the x-class

value one or more CSS class to apply.

So, in other words, the x-class is a name of a group of CSS classes to be applied to a particular x-tag.
