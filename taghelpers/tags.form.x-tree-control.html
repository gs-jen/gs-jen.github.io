tag x-tree-control in Form

Brief

The tree control.

Declaration
<x-tree-control id="text" var="text" height="text" width="text" />

Parameters
id

The tree identifier.
var

The variable to store associated kendoTreeView object.
height

The height of the control in pixels.
width

The width of the control in pixels or in percents.

Details

The control supports the following events change, select and databound.

The data to the tree view may be provided in two ways:

1) Using a tree of x-tree-item controls.

2) Providing the data from the server.

To setup the server data use x-server-datasource tag, just use x-server-tree-model tag if you want to set a model instead of x-server-model.

To provide the data from the server you need to create an action that takes one integer parameter - the identifier of the parent node. To get the root, the tree view will request the data with no parameters set.

As the response, the method shall return an object with the data field which consists of an array of json object. Each object must consists of at least three properties:

Name

Type

Description

id

int

The node identifier

text

string

The text to be displayed

HasChildren

boolean

The flag indicating whether the node has children nodes.

See also TreeView Overview.

All attributes described in x-form-control base tag can also be applied to this tag.
