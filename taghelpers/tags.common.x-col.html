tag x-col in Common

Brief

A bootstrap layout column

Declaration
<x-col xs-size="number" xs-offset="number" sm-size="number" sm-offset="number" md-size="number" md-offset="number" lg-size="number" lg-offset="number" xs-visible="true|false" sm-visible="true|false" md-visible="true|false" lg-visible="true|false" />

Parameters
*-size

The offset of the column on the corresponding screen size expressed in layout columns (1...12).

Screen sizes stands for: xs extra-small, sm small, md medium and lg large.
*-offset

The offset of the column on the corresponding screen size expressed in layout columns (0...12).
*-visible

The flag indicating whether the tag is visible on corresponding screen size.

Details

All attributes described in x-base base tag can also be applied to this tag.
