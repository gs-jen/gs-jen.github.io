tag x-date-edit in Form

Brief

The edit for date or date and time.

Declaration
<x-date-edit size="number" var="text" include-time="true|false" format="text" show-week-number="true|false" min="date" max="date" interval="number" calendar-start="default|month|year|decade|century" calendar-depth="default|month|year|decade|century" use-formatted-input="true|false" />

Parameters
size

The size of the control in characters.
var

The name of the variable to put kendoDatePicker object.
include-time

The flag indicating whether editor must also edit hours and minutes of the date.
format

The date/time format.

For Kendo date formats see Date formatting in Kendo UI documentation
show-week-number

The flag indicating whether the control must show the week numbers.
min

The minimum date allowed to be selected (1/1/1900 by default).

Please note that the behavior in case the date is earlier than minimal date isn't intuitive, the control simply returns null instead of the value requested. So, it is advised to always pair a minimum date specification with the proper validation.
max

The maximum date allowed to be selected.
interval

The interval between time values to select in minutes. The default value is 15.
calendar-start

The attribute sets an initial drop-down calendar view.

The default is month

The attribute can take one of the following values: default, month, year, decade, century
calendar-depth

The attribute sets the maximum depth of a drop-down calendar view.

The attribute can take one of the following values: default, month, year, decade, century
use-formatted-input

The flag indicating whether a masked edit control with the mask appropriate for chosen date format is used.

Details

The control supports change and disable-date (specify using byname) events).

The disable-date event takes a date as parameter and returns true or false indicating whether the date needs to be disabled.

All attributes described in x-form-control base tag can also be applied to this tag.

See also DatePicker Overview and DateTimePicker Overview
