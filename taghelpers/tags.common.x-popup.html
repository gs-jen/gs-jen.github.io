tag x-popup in Common

Brief

The pop-up window simulation.

Declaration
<x-popup id="text" title="text" content="text" content-height="text" width="text" var="text" buttons="" />

Parameters
id

The pop-up ID
title

The pop-up title.
content

The pop-up content.
content-height

The pop-up height in pixels or percents.
width

The pop-up height in pixels or percents.
var

The variable to place associated kendoDialog object.

Details

To set pop-up text use $('#id').text('text to set').

To show pop-up varName.open();.

To add buttons, place x-button with action type inside x-popup tag.

To close pop-up in button action use e.sender.close();.

See also Dialog Overview.

All attributes described in x-base base tag can also be applied to this tag.
