tag x-menu in Common

Brief

The menu

Declaration
<x-menu logo-image="url" logo-href="url" logo-height="number" menu-id="text" mobile-menu-id="text" menu-name="text" mobile-menu-sprite="text" menu="IMenu" orientation="horizontal|vertical|verticalPanel" width="text" scrollable="true|false" panel-item-class="text" auth-info="IAuthInfo" />

Parameters
logo-image

The image of the logo to put to left-top corner.
logo-href

The URL to navigate when logo image is clicked.
logo-height

The height of the logo in pixels.
menu-id

The menu identifier.
mobile-menu-id

The mobile menu identifier. If this attribute is not set the mobile (aka responsive) menu won't be created.
menu-name

The name of the menu. The name of the menu will be displayed in the mobile version. If the name is not set, only sprite icon will be displayed on the top of th menu.
mobile-menu-sprite

The mobile icon sprite.

See also Kendo UI icons and Bootstrap glyphicons for available icons.
menu

The menu content defined provided via menu interface.

See also IMenu interface.

You can use either x-menuitem-* tags or IMenu to provide the menu content but not both together.
orientation

The menu orientation:

The attribute can take one of the following values:

horizontal - the menu on the top of the page with responsive option.

vertical - the menu along left side of the page.

verticalPanel - the menu made as accordion panel along the left side of the page.
width

The width of the vertical menu in pixel or percents.
scrollable

The flag indicating whether menu is scrollable.
panel-item-class

For the panel menu, the class to apply to an individual accordion panels.
auth-info

The interface that provides information about the current authentication.

If no menu items use when-logged-in or permissions attributes, this attribute may be omitted.

See also IAuthInfo interface.

Details

Only x-menuitem, x-menuitem-submenu, x-menuitem-submenu or x-menuitem-url tags can be used inside of this tag.

All attributes described in x-base base tag can also be applied to this tag.

See also Menu Overview.

How to Make Horizontal Menu Full Width

Add following styles to the page stylesheet:

Style [show]

Style [hide]

.container, .container-fluid {
    padding: 0;
}

Put menu within a div that has styles container-fluid k-content col-xs-12

Add style container-fluid to the x-menu tag.

Make Last Item Menu of Horizontal Adjust to Right

Add following styles to the page stylesheet:

Style (Replace menu with id of your menu) [show]

Style (Replace menu with id of your menu) [hide]

#menu > .k-last {
    float: right;
    border-left-width: 1px;
    border-right-width: 0;
}

What If Spites Aren't Displayed

Add following styles to the page stylesheet:

Style [show]

Style [hide]

.k-sprite {
    text-indent: 0;
    font-size: 16px;
    height: 24px;
    line-height: normal;
}
