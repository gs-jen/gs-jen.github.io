tag x-static in Form

Brief

The non-editable static text control.

Declaration
<x-static messages="string[]|IEnumerable<string>" static-type="static|success|information|error" icon="iconclass" />

Parameters
messages

The attribute message is used when multiple text lines are desired to be specified as static message. The value of this attribute may be either IEnumerable<string> or string[]
static-type

The attribute defines the look of the static control. It will look as a text or will be a colored full-width information block. The default value of this attribute is static.
icon

The attribute defines the icon to be displayed.

See also Kendo UI icons and Bootstrap glyphicons for available icons.
auto-hide

The attribute defines the tag must be hidden in case there is no tag content.

The attribute is false by default for a static and true by default for block types.

Details

The static content is made of (in the order specified):

  • icon

  • value or messages text

  • x-static tag content text

All attributes described in x-form-control base tag can also be applied to this tag.
