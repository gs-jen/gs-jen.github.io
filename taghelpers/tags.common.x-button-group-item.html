tag x-button-group-item in Common

Brief

One item inside a button group.

Declaration
<x-button-group-item text="text" badge="text" icon-sprite="text" icon-image="text" enabled="true|false" />

Parameters
text

The text on the button.
badge

The badge on the button.
icon-sprite

The class of a sprite for the icon.

See also Kendo UI icons and Bootstrap glyphicons for available icons.
icon-image

The URL of an image for the icon.
enabled

The flag indicating whether the item is enabled.

Details

The tag may be used inside of x-button-group tag only.
