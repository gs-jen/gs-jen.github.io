What is CSR (client-side rendering)?

CSR is a set of functions intended to create dynamic content of x-form on the client side.

In order to add controls at run time get 'csr_layout' data property of the form element and read layout property:
var layout = $('#form1').data('csr_layout').layout;

Please note that CSR supported in forms in horizontal or twocolumn mode only.
