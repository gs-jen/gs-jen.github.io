tag x-listbox in Form

Brief

The listbox control.

Declaration
<x-listbox options="object" draggable="true|false" var="text" connect-with="text" post="none|selection|content" validator-get-value="text" ajax-set-value="text" />

Parameters
options

This attribute is used only when the post mode of the tag is none or selection.

The attribute value must be the programmatic data provider for the options which either

1) must implement a ISelectSource interface.

or

2) must be an array or a list of objects which are marked up with proper JsonPropetry attributes and defines at least two json attributes: value and text.
draggable

The flag indicating whether the content of the list box may be dragged.

The x-listbox-dropsource to define the listbox which is the source of the drag and drop operation.
var

The name of the variable to store kendoListBox object.
connect-with

The identifier of listbox to be connected to (for handling toolbar-defined operations), see x-listbox-toolbar.
post

The mode on what the listbox control must post as the part of the form (or save to javascript object for ajax forms).

The attribute can take one of the following values:

none - the control posts nothing.

selection - the control posts the value of the currently selected item.

content - the control posts the array of value and text pairs of all object inside the listbox

Details

The control supports change, template, drag.placeholder, drag.hint, drag.start, drag.end, drag, drop, add, reorder, remove (all except the first two are specified as byname) events).

The tag may consists of the following tags:

x-listbox-dropsource, x-listbox-toolbar.

See also ListBox Overview and JQuery API for ListBox.

All attributes described in x-form-control base tag can also be applied to this tag.
