class RecurrenceRule

Brief

The recurrence rule.

Details

The recurrence rule is the presentation of the recurrence rule as it is described in section 3.3.10 of RFC 5545 - Internet Calendaring and Scheduling Core Object Specification.

Please refer to RFC for details on meaning and proper interpretation of the recurrence rule properties.

The type defined in the Gehtsoft.HtmlHelpers.dll assembly. The namespace is Gehtsoft.TagHelpers.
