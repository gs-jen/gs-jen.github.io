static public method Color.FromRgba

Brief

Creates a Color structure from the four RGBA components (red, green, blue and alpha) 0.0 - 1.0 values.

Declaration
C#
Gehtsoft.PDFFlow.Models.Shared.Color  FromRgba (double r, double g, double b, double a = 1)

Parameters
r

The red component value of the color.
g

The green component value of the color.
b

The blue component value of the color.
a

The alpha component value of the color. This parameter is optional. By default, the value is 1.0 which means that the color is fully opaque. The value 0.0 means that the color is fully transparent.

Details

The components must have values between 0.0 (0%, 0 byte value) and 1.0 (100%, 255 byte value).
