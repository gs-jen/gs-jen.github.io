WrongScalingMode

Exception type:
ArgumentException

May occur:
When adding an image with ScalingMode.OriginalSize or ScalingMode.Stretch.

Cause:
The image size is specified. When the scaling mode is ScalingMode.OriginalSize or ScalingMode.Stretch, the image size is defined automatically. The image size can be specified in the following scaling modes: UserDefined or None.

ImageExtensions.AddImage(this SectionBuilder, byte[], XSize, ScalingMode),

ImageExtensions.AddImageToSection(this SectionBuilder, string, XSize, ScalingMode),
ImageExtensions.AddImage(this SectionBuilder, byte[], float, float, ScalingMode),
ImageExtensions.AddImage(this SectionBuilder, string, float, float, ScalingMode),
ImageExtensions.AddImageToSection(this SectionBuilder, string, float, float, ScalingMode),
ImageExtensions.AddImageToSection(this SectionBuilder, byte[], float, float, ScalingMode),
