public method TableBuilder.SetContentRowStyleBackColor

Brief

Sets the background color for the content row style.

Declaration
C#
Gehtsoft.PDFFlow.Builder.TableBuilder  SetContentRowStyleBackColor (Gehtsoft.PDFFlow.Models.Shared.Color color)

Parameters
color

The background color.
The color can be specified using a system-defined color (for example, Color.Yellow), can be translated from an HTML color code using the method Color.FromHtml, or created from RGBA values using the method Color.FromRgba.
The RGBA red, green, blue, and alpha components must have values between 0.0f (0%, 0 byte value) and 1.0f (100%, 255 byte value). The alpha component is optional, its value is 1.0f by default, which means that the color is fully opaque.

Details

This method allows you to set the background color for all content rows of the table. If no style is applied to the content rows, the library applies the default content row style and applies the specified background color to this style. If some style is applied to the content rows, the library applies the specified background color to this style.
