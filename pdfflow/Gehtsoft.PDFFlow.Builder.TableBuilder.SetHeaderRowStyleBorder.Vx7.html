public method TableBuilder.SetHeaderRowStyleBorder

Brief

Sets the border for the header row style by a configuration action.

Declaration
C#
Gehtsoft.PDFFlow.Builder.TableBuilder  SetHeaderRowStyleBorder (System.Action<Gehtsoft.PDFFlow.Builder.BorderBuilder> configure)

Parameters
configure

A configuration action for the borders

Details

This method allows you to configure the border for all header rows of the table. If no style is applied to the header rows, the library applies the default header row style and applies the specified border to this style. If some style is applied to the header rows, the library applies the specified border to this style.
