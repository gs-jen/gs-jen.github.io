public method PriceArchiveClient.GetAllRanges

Brief

Gets the ranges of the data for all the instruments available.

Declaration
C#
InstrumentRange[]  GetAllRanges ()
C#
Task<InstrumentRange[]>  GetAllRangesAsync ()

Details

The method return the list of the instruments, timeframes and associated ranges of data for a particular instrument in a particular timeframe.

Note: if a group of timeframes has the same range, e.g. 1m, 5m, 15m and so on, only the shortest time frame will be represented in the list.

E.g. if the answer consists of 1m and 1h timeframes only, that means that all timeframes longer than 1m and shorter than 1h (1m, 5m, 15m, 30m) will have one range of the data, and all timeframes longer than 1h (1h, 2h, 6h, 12h, 1d...) will have the other range.

Example

var dataCollections = client.GetAllRanges();
foreach (var dataCollection in dataCollections)
    Console.WriteLine("{0} {1} {2}-{3}", dataCollection.Instrument, dataCollection.Timeframe.ToString(), dataCollection.Range.From, dataCollection.Range.To);
