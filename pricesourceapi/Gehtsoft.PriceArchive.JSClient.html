class Gehtsoft.PriceArchive.JSClient

Brief

JavaScript client for the PriceArchive API

Details

To use the JavaScript client API add a reference to API definition into your code:

To add reference to JavaScript API in HTML

<script src="https://pricearchive.gehtsoft.com/lib/Gehtsoft.PriceArchive/Gehtsoft.PriceArchive.JSClient.min.js?version=1.2.4" />

To add reference using bower

.bowerrcCommand line
{ "registry": "https://www.myget.org/F/gehtsoft-public/bower/" } 
.bowerrcCommand line
$bower install Gehtsoft.PriceArchive.JSClient

To use package in node.js

.npmrcCommand lineCode
registry=https://www.myget.org/F/gehtsoft-public/npm/ 
.npmrcCommand lineCode
$npm install price-archive@1.2.5
.npmrcCommand lineCode
let JSClient = require('./node_modules/price-archive/Gehtsoft.PriceArchive.JSClient.min.js');
let client = new JSClient('https://pricearchiveapi.gehtsoft.com');

To start using the client in your code create an instance of the object:

let client = new Gehtsoft.PriceArchive.JSClient("https://pricearchive.gehtsoft.com");
