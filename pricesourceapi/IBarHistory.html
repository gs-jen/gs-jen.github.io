class IBarHistory

Brief

A history of the price bars

Details

The class is a collection of price bars

Public Methods

find

Finds a bar which consists of the date and time specified.

getAskClose

Returns the ask close price of the bar specified

getAskHigh

Returns the ask high price of the bar specified

getAskLow

Returns the ask low price of the bar specified

getAskOpen

Returns the ask open price of the bar specified

getAskVolume

Returns the ask volume of the bar specified

getBidClose

Returns the bid close of the bar specified

getBidHigh

Returns the bid high price of the bar specified

getBidLow

Returns the bid low price of the bar specified

getBidOpen

Returns the bid open price of the bar specified

getBidVolume

Returns the bid volume of the bar specified

getCount

Returns the number of bars.

getInstrument

Returns the name of the instrument.

getPipSize

Returns a size of a pip (the minimum price change).

getPrecision

Returns the number of significant digits after the decimal point.

getStartDate

Returns the start date of the specified bar

getTimeframe

Returns the length of one bar in the collection.
