public method Gehtsoft.PriceArchive.JSClient.GetPrices

Brief

Returns the price history

Declaration
JavaScript
Gehtsoft.PriceArchive.JSClient.GetPrices (callback, instrument, timeframe, from, to)

Parameters
instrument

The name of the instrument.
timeframe

The timeframe (in from of "Nt" where N is the length, and t is the timeframe type, e.g. 1m).

Following timeframes are supported: 1m, 2m, 3m, 5m, 6m, 10m, 15m, 20m, 30m, 1h, 2h, 3h, 4h, 6h, 12h, 1d, 1m, 1y
from

The date and time to get the prices from
to

The date and time to get the prices to
callback

The callback will be called when the method is complete and must be a function accepting an object which has the following properties:

Code

The error code or 0 if the operation was successful

Error

The error message in case the operation wasn't successful

Bars

An array of the objects representing the price bars

See below for the bar object description.

A history is represented as the object with the following properties

Instrument

The instrument name.

Timeframe

The name of the timeframe

Precision

The number of significant digits after the decimal point

PipSize

The minimum change of the price

Note: don't be confused by the description, some markets have sub-pip price change, e.g. 1/10 of a pip.

Bars

An array of the objects representing the price bars

See below for the bar object description.

Each bar is represented as the object with the following properties:

Start

The date and time when the bar started.

Bid.Open

Open (the first) bid price of a bar.

Bid.High

Highest (maximum) bid price of a bar.

Bid.Low

Lowest (minimum) bid price of a bar.

Bid.Close

Close (the last) bid price of a bar.

Bid.Volume

The volume of the trades.

Ask.Open

Open (the first) ask price of a bar.

Ask.High

Highest (maximum) ask price of a bar.

Ask.Low

Lowest (minimum) ask price of a bar.

Ask.Close

Close (the last) ask price of a bar.

Ask.Volume

The volume of the trades.

Details

Example:

Get the price data

client.GetPrices(function (result) {
   if (result.Code == 0) {
       console.log(result.History.Instrument, result.History.Timeframe, result.History.Precision, result.History.PipSize);
       for (let i = 0; i < result.History.Bars.length; i++) {
          let bar = result.History.Bars[i];
          console.log(bar.Start, bar.Bid.Open, bar.Bid.High, bar.Bid.Low, bar.Bid.Close, bar.Ask.Open, bar.Ask.High, bar.Ask.Low, bar.Ask.Close);
      }
   } else {
       console.log("Can't get instrument data");
       console.log(result.Message);
   }
}, "EURUSD", "1m", "2010-09-01 00:00", "2010-09-02 00:00");
